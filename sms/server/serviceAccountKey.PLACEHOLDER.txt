IMPORTANT — serviceAccountKey.json
====================================

This file is a placeholder. You must replace it with your real
Firebase service account key before running the backend server.

Steps to get your key:
1. Go to https://console.firebase.google.com
2. Open your project → Project Settings (gear icon)
3. Click the "Service Accounts" tab
4. Under "Firebase Admin SDK", click "Generate new private key"
5. Save the downloaded .json file as:
       server/serviceAccountKey.json
   (replacing this placeholder file)

⚠️  SECURITY WARNING:
   Never commit serviceAccountKey.json to version control.
   It is already listed in .gitignore for your protection.

The downloaded file will look like this:
{
  "type": "service_account",
  "project_id": "YOUR_PROJECT_ID",
  "private_key_id": "...",
  "private_key": "-----BEGIN RSA PRIVATE KEY-----\n...",
  "client_email": "firebase-adminsdk-xxxxx@YOUR_PROJECT.iam.gserviceaccount.com",
  "client_id": "...",
  "auth_uri": "https://accounts.google.com/o/oauth2/auth",
  "token_uri": "https://oauth2.googleapis.com/token",
  ...
}
